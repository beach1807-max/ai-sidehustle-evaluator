# 修正付費預覽區塊文案：從深度報告改為 AI Agent 開工包

## 任務目標

目前付費預覽區塊的文案仍停留在舊版本，例如：

* 深度報告預覽
* Codex 開發提示詞
* 產生深度報告
* 單次深度報告：NT$49～99

但目前產品已經演進為：

* Deep Report
* AI Agent 開工包
* 給 AI 的完整開發任務說明
* MVP 開發說明書
* 推廣文案
* 收費建議
* 風險修正與砍功能建議

本次任務是修正付費預覽區塊文案，讓它符合目前產品價值。

## 文案調整

### 1. 標籤文字

將：

```text
深度報告預覽
```

改成：

```text
AI Agent 開工包預覽
```

---

### 2. 主標題

將：

```text
想把這個點子變成可執行 MVP？
```

改成：

```text
想讓 AI 幫你做出第一版網站？
```

---

### 3. 說明文字

將原本類似：

```text
解鎖副業點子深度驗證報告，取得更具體的 7 天執行方案與 Codex 開發提示詞。
```

改成：

```text
系統會把你的副業點子整理成一份 AI 看得懂的開發說明書，包含 MVP 功能、7 天行動計畫、AI Agent 開工包、推廣文案與收費建議。
```

---

### 4. 右側清單標題

